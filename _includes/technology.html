Technology

Built on the same tools used by museum researchers.

Positioning Niantic Lightship VPS
Visual Positioning System that anchors AR content to real-world locations with centimeter-level accuracy — the same platform behind large-scale public AR deployments.
Site capture Scaniverse + LiDAR
On-site photogrammetry and LiDAR scanning conducted at each mural using USF Access 3D Lab equipment, producing precise environmental models for AR anchoring.
3D & interaction Unity, C#
Each mural is a self-contained Unity scene routed from a lightweight bootstrapper. Animations, audio, shaders, and depth-mask effects all live inside the AR engine.
Android frontend Kotlin, Jetpack Compose
The Android shell is a modern Compose application that hands off to Unity for each AR scene, keeping the app light on devices without flagship hardware.
Map OpenStreetMap
Migrated from Google Maps to eliminate billing dependencies — a civic project deserves a civic map. Custom overlays surface Bartow businesses and mural sites.
